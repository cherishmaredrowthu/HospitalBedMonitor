About BedMonitor Live
Project Goal

To reduce life-saving response time by providing citizens with real-time, location-aware data on hospital bed availability.

How it Works
  • Location Sorting: Automatically identifies your coordinates to suggest the nearest help.
  • Staff Verification: Bed counts are managed by authenticated hospital staff.
  • Routing: Get live distance and time estimates for emergency transit.
Tech Overview

Backend: Java Spring Boot, MySQL
Mapping: Leaflet JS & OpenStreetMap
Geolocation: Real-time browser GPS tracking
